Политика касаемо обработки персональных данных

1. Общие положения

Настоящая политика обработки персональных данных (далее — «Политика») определяет порядок и условия обработки персональных данных, предпринимаемые командой Umbra-Grid (далее — «Оператор»), и устанавливает меры по обеспечению безопасности этих данных в рамках предоставления доступа к инновационным технологиям.

1.1. Важнейшей целью Оператор считает соблюдение прав и свобод человека при обработке его персональных данных, обеспечивая прозрачность использования AI-инструментов для реальных задач и защиту частной жизни.

1.2. Настоящая Политика применяется ко всей информации, которую Оператор может получить о посетителях (далее — «Пользователи») веб-сайта umbra-grid.blog.

2. Основные понятия, используемые в Политике

  • Веб-сайт — совокупность графических материалов, 3D-сеток и информационных ресурсов, доступных по адресу umbra-grid.blog.
  • Пользователь — любой посетитель, изучающий практики применения ИИ на нашем ресурсе.
  • Персональные данные — любая информация, предоставленная Пользователем для получения экспертной поддержки или доступа к платформе.
  • Обработка персональных данных — любые действия с данными, включая сбор для настройки персональной стратегии роста на базе инноваций.
  • Безопасность — комплекс мер, направленных на защиту данных в условиях передовой цифровой инфраструктуры.

3. Данные, которые мы обрабатываем

Оператор может обрабатывать следующие категории данных:

  • Данные, предоставляемые Пользователем:
    • Имя и фамилия;
    • Адрес электронной почты (Email) вида support@node-onix.today или hello@umbra-grid.blog;
    • Номера телефонов (только цифры).
  • Данные, собираемые автоматически:
    • Обезличенные данные о посетителях (файлы «cookie»), собираемые для улучшения взаимодействия с интерактивными элементами сайта и анализа эффективности методологий.

4. Цели обработки персональных данных

  • Идентификация Пользователя для открытия доступа к платформе и базе знаний.
  • Установление обратной связи для проведения консультаций по всем вопросам применения ИИ.
  • Информирование о новых решениях, которые меняют правила игры, и программах пассивного дохода.
  • Отказ от получения уведомлений возможен в любой момент через письмо на hello@umbra-grid.blog с пометкой «Отказ от уведомлений».
  • Оптимизация работы сайта для удобной адаптации пользователей без опыта.

5. Правовые основания обработки

Оператор обрабатывает данные только при наличии добровольного согласия Пользователя, выраженного через заполнение форм регистрации или подписки на сайте. Мы действуем в рамках европейских стандартов защиты данных (GDPR).

6. Порядок сбора, хранения и безопасности данных

  • Мы принимаем все необходимые технические меры, чтобы ваша работа с технологиями нового поколения оставалась безопасной.
  • Персональные данные Пользователя никогда не передаются третьим лицам, за исключением случаев выполнения требований законодательства ЕС и Италии.
  • Срок обработки данных неограничен, но Пользователь может отозвать согласие, направив уведомление на hello@umbra-grid.blog.

7. Заключительные положения

  • Актуальная версия Политики всегда доступна в свободном доступе на сайте umbra-grid.blog.
  • По всем вопросам, касающимся вашей конфиденциальности в мире ИИ, обращайтесь к нам по адресу: hello@umbra-grid.blog.
